Authentification: Getting an API Key

Before you begin, you should sign up for a NASA API key.


After signing up, you will recieve an email with your key which you will use in this tutorial. As with most other APIs, it is necessary to request a key so that the organization hosting the API can keep track of who is requesting its data. The host can also limit the amount of requests made by a single application or person so as to limit the burden on its servers. For the NASA APIs, you are allowed 1000 requests per hour.
